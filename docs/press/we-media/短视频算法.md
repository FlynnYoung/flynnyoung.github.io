# 通用短视频算法

最近在微信上看到一个关于短视频算法的文章，因为我本身打算开始转行做自媒体，感觉这篇文章很有价值，这里记录一下。

## 前言

文章写的是一个印度19岁少女 @Jenny Hoyos,通过分析 @MrBeast 等大V 的视频，总结出的通用短视频算法。在此算法的基础上，她的视频在全网狂揽 6 亿播放量。个人觉得还是很有必要学习并记录一下💪。


## 算法内容
文章中@Jenny Hoyos 总结了三招
### 第一招： 病毒引擎 -“故事+反转” 公式
首先做短视频要先忘掉那些花里胡哨的技巧，核心秘诀只有两个词：故事和反转。任何一个无聊的字体，比如“画画用的颜料”，只要注入一个引入入胜的故事，就能让观众欲罢不能。

::: tip 举例说明
比如拍一个“为陌生人做饭”的视频，这听起来很普通，但是如果加入个人故事：“我的厨房坏了，我需要通过为别人做饭来赚钱修理它”
:::

看到了吗？一个充满“讽刺感”的个人困境，瞬间让视频有了灵魂。观众不在是旁观者，而是情感的参与者。这就是病毒传播的起点：`用一个强有力的“为什么”，让观众产生情感共鸣`。

### 第二招： 黄金法则-精确到秒的“34秒完美结构”

@Jenny Hoyos 通过海量数据分析发现，她最火的视频，时长都惊人的集中在 34 秒！这并非巧合，而是一个精心设计的叙事框架。

她的每一个视频都严格遵循这个结构：

::: warning 1. 致命钩子（前3秒）
钩子必须是高度视觉化的，就算静音也能看懂。更绝的是，她会用“可读性检查器”来确保文案简单到五年级的学生都能理解。她甚至分析了@MrBeast 所有视频的脚本，发现其可读性只有一年级水平！这就是为什么他们的视频能病毒式传播。
:::

::: warning 2. 悬念铺垫（3-5秒）
在钩子之后，立刻用一句话告诉观众视频结尾会发生什么。比如，“我要去海边，并在最后给一个陌生人100美元的惊喜。” 这给了观众一个看下去的明确理由。
:::

::: warning 3. 丝滑过渡
绝不说“让我们开始吧”这种废话！她会用一句承上启下的话来保存节奏。比如：“麦当劳禁售商品”的视频里，她的过渡是：“所以我非法的烹饪了它”。即是过渡，又是新的悬念。
:::

::: warning 4. “但是/因此”主体
用不断的故事转折来推动故事。“我出门散步，但是下雨，因此我跑回家...”这种叙事方式能牢牢抓住观众的注意力。
:::

::: warning 5. 惊喜结尾
在观众期待的结局上，加上一个意想不到的反转。就像她给妈妈的礼物，满足了观众的期待，但最后礼物却被摔坏了，这成了神来之笔。
:::

### 第三招： 数据为王-“一秒钟”拯救一个视频
@Jenny Hoyos对数据的痴迷近乎疯狂。她分析了一个真实案例：一个视频发布后表现平平，留存率只有83%。她打开留存图标，发现最后一秒，数据从70%断崖式下降到45%。

她果断剪掉了这多余的1秒。就这一秒钟，视频留存率飙升到88%，播放量瞬间起飞！

她得出一个惊人的结论

> [!IMPORTANT]
>短视频的每一秒都至关重要，90%以上的留存率是爆款的最低门槛！而这个数字往往是通过结尾的巧妙设计让观众“循环播放来实现”

## 最后的颠覆性思考
- 平台不是同一个宇宙，有的喜欢故事、有的偏爱快节奏信息、有的需要强视觉和字幕。因地制宜才是王道

- 留存率也并非唯一真神：很多留存率100%的视频并没有大火，而一些低留存视频却能爆火。算法最终衡量的，可能不是冷冰冰的数据，而是一个更玄学的指标——观众满意度。

## 总结

- 一个核心： 用“故事+反转”注入灵魂
- 一个法则： 遵循“34秒完美结构”
- 一个信用： 数据为王，抠到每一秒
- 一个洞察： 观众满意度高于一切
